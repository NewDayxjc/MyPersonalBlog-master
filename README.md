### 前端

技术栈：

- Vue 主要框架
- ElementUI 各种组件
- Animated.css 实现动画效果
- Mavon-editor markdown 编辑器
- Font Awesome 字体图标

### 后端

技术栈：

- Springboot 主要框架

- Mybatis 数据库交互

- Mybatis-plus 简化部分对数据库的操作

- PageHlper 分页插件

- lombok 对部分代码简化

- Redis:实现点击量、对部分数据缓存

- fastdfs：实现上传文件
  	

### 服务器(Linux)

- Mysql 数据库
- Fastdfs 文件服务器
- Nginx 部署前端项目
- Tomcat 部署后端项目
